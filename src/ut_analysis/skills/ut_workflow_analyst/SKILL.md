# ut-workflow-analyst Skill

## Purpose

This specialist analyzes workflow issues, process breakdowns, and flow failures within usability findings for the lane.
It groups related findings into themes, summarizes supporting evidence, and generates
lane-specific product implications for downstream thematic synthesis.

## Focus Areas

- Theme detection from shared keywords in finding descriptions
- Evidence rollups across participants and tasks
- Representative quotes and outlier detection
- Product implications tailored to the lane perspective

## Output

The skill returns a structured lane analysis with per-theme evidence counts,
participant coverage, contradictions or outliers, confidence, and limitations.
