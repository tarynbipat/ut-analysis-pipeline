# ut-run-summary Skill

Generates a pipeline run summary artifact that makes the workflow inspectable.

## What it shows

- stages completed
- artifacts created
- agents invoked
- evals passed and failed
- review checkpoints created
- next suggested actions

## Purpose

This skill improves observability across the pipeline by creating a clear run summary artifact that helps people understand what happened, what was produced, what still needs attention, and what to do next.
